# Audit Scope

## Audit Overview

SecureShop SA commissioned a compliance assessment to evaluate the effectiveness of controls supporting PCI-DSS v4.0 and POPIA requirements.

## Systems Included

The following systems were in scope:

### Cloud Infrastructure

* Azure Subscription
* Azure Resource Group
* Azure SQL Database
* Azure Storage Accounts
* Azure Key Vault
* Azure App Services

### Identity Systems

* Microsoft Entra ID
* Conditional Access
* Privileged Identity Management

### Monitoring Systems

* Microsoft Sentinel
* Log Analytics Workspace
* Microsoft Defender for Cloud

### Business Applications

* SecureShop E-Commerce Website
* Customer Support Portal
* Administrative Portal

## Data Included

The assessment covered:

### Personal Information

* Customer Names
* Email Addresses
* Telephone Numbers
* Physical Addresses

### Sensitive Information

* Identity Numbers
* Payment References
* Authentication Data

### Cardholder Data

* Payment Transactions
* Tokenized Payment Information

## Exclusions

The following were excluded:

* Third-party payment processor infrastructure
* Bank infrastructure
* End-user devices
* Physical site assessments

## Assessment Period

Audit Period:
01 January 2026 – 31 January 2026

Frameworks:

* PCI-DSS v4.0
* POPIA
